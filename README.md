# MyFlix 
## ( Project is currently WIP )

This project is a my attemp to clone the basic functionalities and UI of Netflix. This project will be using the [The Movie Database](https://www.themoviedb.org/) API to fetch movies and their data, the other following libraries I have used so far are listed below. Will add more in the future as project is still ongoing.

### Tech stack used so far
- [Create React App](https://create-react-app.dev/docs/getting-started/)
- [React Router](https://reactrouter.com/)
- [Styled components](https://styled-components.com/)
- [React Icons](https://react-icons.github.io/react-icons/)
- [Axios](https://github.com/axios/axios)
- [Firebase](https://firebase.google.com/)
